# Nuedae Outfit Tool

A Python tool built as part of the Nuedae fashion app project. A sustainable outfit generator that eliminates decision fatigue by building looks from what you already own.

## What it does
- Stores outfits as structured data (name, category, rating)
- Loops through all outfits and prints a summary
- Handles missing fields gracefully without crashing
- Exports the full outfit list to Excel using pandas

## Tools Used
- Python
- pandas
- Jupyter Notebook

## About Nuedae
Nuedae is a fashion app concept that generates outfits from your existing closet, builds community around personal style, and promotes sustainability by encouraging you to buy less and wear more.
